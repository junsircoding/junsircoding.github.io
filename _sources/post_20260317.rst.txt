用 QGIS 手动修改 Shapefile
===========================

读取矢量数据
------------------------------

在绘制地图时，常用的地图边界数据格式有 ``shp`` 、 ``gpkg`` 、 ``geojson`` 等，当把它们用 QGIS 读取时，其实是同一种矢量数据。

用以下步骤加载源文件：

.. code-block::
    :linenos:

    Layer - Add Layer - Add Vector Layer

这时加载进来的数据会像 PhotoShop 一样有一个图层。

我们现在的这个地图数据有一点问题，在越南和广西的交界处有一点空隙：

.. figure:: /_images/20260317/相邻边界有一个缺口.png
   :alt: 相邻边界有一个缺口

现在我们想要把这个缺口手动处理掉，让这个边界看起来更自然。


手动修复问题图层
------------------------------

右键这个图层，选择 ``Toggle Editing`` 可以把图层变成可以编辑的模式。

在成功转为可编辑模式之后，菜单上面有一个 ``Vertex Tool`` 就从原来的不可用（灰色）变成可以使用（彩色），选择 ``Vertex Tool (Current Layer)`` 。

.. figure:: /_images/20260317/VertexTool图标.png
   :alt: VertexTool图标

在目前读入的这个地图中，越南是其中的一个 Feature。
这时用鼠标点一下越南这个 Feature ，就会显示出这个多边形的每个顶点。

.. figure:: /_images/20260317/编辑模式.png
   :alt: 编辑模式

用鼠标点某个顶点，接着把鼠标移到新位置再点一下，这个点位置就更新了。按照这样的步骤，通过移动顶点，来改变整个形状。

手动去把边界调整成想要的不规则形状比较麻烦，我们的思路是先把越南的部分“侵入”广西，然后沿着广西的边界对越南进行裁剪。

.. figure:: /_images/20260317/将越南稍微侵入广西一点.png
   :alt: 将越南稍微侵入广西一点

把扩大后的越南用 SQL 单独筛选出来：

.. code-block::
    :linenos:

    SELECT * FROM "province.v0" WHERE "国家" = '越南'

    # 表名就是图层名字, 字段名加双引号

我们把它命名为“越南侵入”。

这时需要先把修改后的越南导出另存为为新的 Feature，因为现在的图层是不能编辑的。
步骤是：

.. code-block::
    :linenos:

    Export - Save Features As

导出之后，他会加载一个新图层，我们将其命名为“越南侵入Feature”。

用同样的方法，把广西也导出来，命名为“广西壮族自治区”。

这时的“越南侵入Feature”，它本身有自交或者不闭合的环，进行后面的裁剪操作会报错。所以需要修复一下。

在左下角的快捷搜索框中搜索 ``Fix`` ，选择 ``Fix geometries`` 。

.. figure:: /_images/20260317/修复gemo.png
   :alt: 修复gemo
    

下面就可以裁剪了，使用 Difference 算法，沿着广西裁剪越南。

.. code-block::
    :linenos:

    Vector - Geoprocessing Tools - Difference

Input 图层选择“越南侵入Feature”，Overlay 图层选择“广西壮族自治区”。

裁剪完之后，两块区域的边界就没有重合了，将裁剪后的结果命名成“越南修正”。


.. figure:: /_images/20260317/越南修正.png
   :alt: 越南修正


替换问题图层
------------------------------

下面把新的越南替换到整个数据里，我们的思路是把新的越南和原数据提出越南的剩余部分合并。

先用 SQL 从原数据里把越南过滤掉。


.. code-block::
    :linenos:

    SELECT * 
    FROM "province.v0" 
    WHERE "国家" != '越南' OR "国家" IS NULL

    # 如果其他feature的被查询字段是NULL,需要手动保留
    # 任何值和 NULL 做比较（=、!=、<>）结果都是 UNKNOWN，而不是 TRUE


得到的图层命名为“原数据去掉越南”。


然后把“原数据去掉越南”和“越南修正”合并：


在左下角快捷搜索栏中搜索 ``Merge vector layers`` ，选择要合并的两个图层。

合并后的图层就是做好的结果了，检查无误之后就可以导出成需要的格式了。
